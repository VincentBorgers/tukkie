# Voice Models

Plaats lokale open-source voice-modellen hier.

- Vosk Nederlands STT: zet het uitgepakte model in `config/models/vosk-model-small-nl-0.22`
- Piper TTS: zet binary en stemmodel in `config/models/piper/`

Deze bestanden zijn bewust uit versiebeheer gehouden. De backend leest de paden uit `.env`.
