# Security Policy

## Ondersteunde release

De actuele ondersteunde release is `v1.0.0`.

## Responsible disclosure

Meld kwetsbaarheden niet publiekelijk via issues. Gebruik in plaats daarvan een directe melding aan:

- Vincent Borgers <vincentborgers2015@gmail.com>

Beschrijf waar mogelijk:

- getroffen component of bestand
- impact en risico
- reproduceerbare stappen
- eventuele mitigatievoorstellen

## Projectprincipes

- Geen directe uitvoering van onbekende of gegenereerde code
- Kritieke acties via bevestigingslaag
- Lokale opslag als standaard
- Secrets, modellen en runtime artifacts buiten versiebeheer
