# Tukkie Open Source Release v1.0.0

Publisher: WebTukker Labs  
Maintainer: Vincent Borgers

## Samenvatting

v1.0.0 is de eerste publieke open-source release van Tukkie. Deze versie zet de basis neer voor een lokale, Nederlandstalige en uitbreidbare home intelligence omgeving die bruikbaar is voor self-hosters, smart home bouwers en open-source bijdragers.

## Belangrijkste punten

- generieke productbranding zonder persoonsgebonden runtime defaults
- schone repository zonder live demo-data als standaardconfiguratie
- gebruikersgerichte README en governance-documentatie
- lokale AI-, memory-, tools- en integration-architectuur
- dashboard voor chat, status, netwerk, energie en voice
- release-hygiëne voor publieke distributie op GitHub

## Wat zit er in deze release

- FastAPI backend
- React + Tailwind dashboard
- conversation memory en long-term memory
- tool permission layer
- lokale voice-architectuur via Vosk en Piper
- integratietemplates voor Tuya, Ring en Imou
- YAML-configuratie voor gebruikersomgevingen

## Open-source positionering

Tukkie is opgezet als productbasis, niet als persoonlijke installatie. Daardoor worden configuratie, live data en secrets losgetrokken van de codebase en blijft de repository klaar voor publieke samenwerking.
