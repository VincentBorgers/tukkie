# Changelog

All notable changes to Tukkie will be documented in this file.

## [1.0.0] - 2026-03-07

### Added

- Open-source branding and repository governance files
- Uitgebreide gebruikersgerichte README
- Apache 2.0 licensing, NOTICE and maintainer metadata
- Schone runtimeconfig met lege live YAML-bestanden en aparte voorbeeldbestanden
- CI workflow voor Python compile-checks en dashboard builds
- Release notes voor de eerste publieke release

### Changed

- Productnaam en user-facing copy omgezet naar Tukkie
- Demo- en persoonsgebonden defaults verwijderd uit runtime en configuratie
- Wake word, profieldefaults en storagepaths generiek gemaakt
- Dashboardteksten en metadata opgeschoond voor publieke distributie

### Security

- Repository blijft vrij van databases, voice-modellen, caches, tokens en lokale secrets
